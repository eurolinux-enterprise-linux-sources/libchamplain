ChamplainImageRenderer

ChamplainImageRenderer — A renderer that renders tiles from binary image data

Description

ChamplainImageRenderer renders tiles from binary image data. The rendering is performed using GdkPixbufLoader so the set of supported image formats is equal to the set of formats supported by GdkPixbufLoader.
